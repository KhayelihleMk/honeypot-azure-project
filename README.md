# Honeypot Azure Lab – Security Detection Project

## Project Overview

This project demonstrates a honeypot deployed on an Azure Windows VM to attract and detect attacker activity. Security logs from the honeypot are forwarded to Microsoft Sentinel for analysis, alerting, and visualization. The lab simulates SOC-level monitoring and provides hands-on experience with cloud-based threat detection.



## Objective

- Deploy a vulnerable Windows VM as a honeypot for research and security monitoring  
- Forward logs to Microsoft Sentinel to detect suspicious activity  
- Analyze and visualize attacker behavior globally  
- Gain hands-on experience with threat detection, log enrichment, and SOC-level alerting


## Skills Demonstrated

- Cloud security monitoring (Azure, Microsoft Sentinel)  
- Honeypot deployment and configuration  
- Log forwarding and enrichment  
- Threat detection and alerting  
- Visualization of attacker activity using map workbooks
